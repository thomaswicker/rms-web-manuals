Download and Installation

 

Running the Trial Version

Trial versions of Sinfonia® for every show can be downloaded at www.rms.biz. The trial version is a fully featured application that allows you to experiment with how Sinfonia® works.  This version gives you full access to the first few songs in the show.  If you decide that Sinfonia® works for you, then you will be able to purchase and authenticate the full version through the program itself, without any additional downloads.

Running the Full Version

Full versions of Sinfonia® are only available to individuals and organizations that have acquired the license for a specific show through the appropriate licensing organization (MTI, R&H, etc). In order to purchase the unlock code for the full version, you will need to license the production through the licensing organization.  The license will run for a particular length of time, depending upon the arrangement you have made with the licensing organization. At the end of your license, Sinfonia® will notify you that your license has expired.

Note: there is no limit to the time you can use Sinfonia® before your performances. So provided you have a valid performance license, you could rehearse with Sinfonia® for as long as you are likely to need it. A few weeks, a month, a year – it's up to you.

Purchasing the Full Version

When you click on the Purchase button, you will be taken to a screen where you will be asked to fill out the appropriate information.  Enter your name, organization, desired show, email address, and location.  Important:  if you have not already licensed the show, you will not be able to purchase the full version.


 

 

 

 

 

 

 

 

 

 

 

If there is an Internet connection issue, then you may receive an error screen. If you then hit “Okay”, a file will be created on your desktop called ‘rms_authentication.txt’. You can email that file to info@rms.biz from a computer with an Internet connection to continue with the purchase request.

Once RMS receives your request, we will check to make sure you have licensed the show for production. Once we have confirmed your license, we will email you a link to an electronic invoice that can be paid online via credit card or e-check. We can also accept paper checks. Please note that we do not accept payment via purchase order. After the payment has been processed, you will be emailed an unlock code.

Authentication

Once you have received your unlock code, you can authenticate the application.  Press the Authenticate button on launch.

You will then be prompted to enter the unlock code from the file you received from RMS.  You may copy and paste this code into the text box (CTRL-C / CTRL-V on Windows, COMMAND-C / COMMAND-V on Mac).  WARNING:  Make sure that you keep the unlock code in a safe place: you will need this in case you wish to reload the application or reset it to original defaults.  The unlock code is machine-specific: you will not be able to use this unlock code on a different computer.  To install on multiple computers, you will need an Additional Organization Computer License.

If you have entered the correct code, then you will receive a confirmation dialog box.  Clicking on OK will the launch the full version of Sinfonia®, and each launch from this point on will bypass the original screen.

Additional Unlock Codes

You are able to install Sinfonia® on up to four computers. This allows you to install the program on computers at your performance location, your rehearsal room, and at home for practice, as well as a license for a backup computer.

Each additional computer will need to request a code through the Purchase or Request Code screen, and after filling out the User Information screen, indicate that you already have a license for the program (provided by your organization) and fill in the license number in the appropriate field. This is not the same as an unlock code. The license for the program is specific to your organization's production, and is provided to the organization on initial purchase.

Once RMS receives your request, you will be emailed an unlock code (provided you have not exceeded the amount of installations permitted by the terms of your License Agreement).

Expiration of License

As mentioned above, the license for a particular show is time based.  If you attempt to launch Sinfonia® after this period, you will be notified of this with a dialogue box.

If you wish to extend the length of the license, or feel that there is a mistake, then please call us at 212-620-0774 and we will be happy to assist you.
